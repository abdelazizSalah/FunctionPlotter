# Function Plotter

## About
* Responsive GUI which plot mathmatical equations on specified range

## 📝 Description
* Takes any arbitrary equation from the user and plot it on given range .
## 🏁 Getting Started

## Dependencies

* Python 
* QTDesigner 

## 💻 Installing

```
git clone https://github.com/abdelazizSalah/FunctionPlotter
```
